## 📌 Description

A modern, interactive number guessing game built using Python and CustomTkinter.
This project combines game logic with a dashboard-style UI, real-time analytics, and data visualization.

---

## 🚀 Features

* 🎨 Modern GUI using CustomTkinter
* 🎯 Difficulty levels (Easy / Medium / Hard)
* 📊 Live stats dashboard (guesses, attempts, games played)
* 📈 Guess trend visualization (Matplotlib chart)
* 🧠 Smart insights and feedback system
* 📜 Guess history tracking
* ⏱️ Real-time timer
* 💾 Best score saved locally
* ✨ Visual effects (color feedback + animations)
* ⌨️ Keyboard support (Enter to submit)

---

## 🛠️ Tech Stack

* Python
* CustomTkinter
* Matplotlib

---

## ▶️ How to Run

### 1. Install dependencies

```bash
pip install customtkinter matplotlib
```

### 2. Run the game

```bash
python game.py
```
